## 📱 手机测试

### 1. 确保手机和电脑在同一WiFi

### 2. 查看电脑IP地址

**Windows:**
```cmd
ipconfig
```
查找 `IPv4 地址`，例如 `192.168.1.100`

**Mac/Linux:**
```bash
ifconfig | grep inet
```

### 3. 手机浏览器访问

在手机浏览器输入：
```
http://[你的IP地址]:8080
```

## 🐛 常见问题

### Q: 页面打不开？
A: 检查端口是否被占用，换个端口试试：
```bash
python -m http.server 3000
```

### Q: 手机访问不了？
A: 
1. 确认电脑防火墙允许8080端口
2. 确认手机和电脑在同一WiFi
3. 尝试关闭电脑防火墙测试

### Q: PWA安装后打不开？
A: 
1. 检查manifest.json路径是否正确
2. 确保所有图标文件存在
3. 清除浏览器缓存重新安装

### Q: 番茄钟倒计时不工作？
A: 打开浏览器开发者工具（F12）查看Console是否有错误

## 📝 开发调试

### Chrome DevTools 移动端模拟

1. 按 `F12` 打开开发者工具
2. 点击左上角手机图标（Toggle device toolbar）
3. 选择设备：
   - iPhone 12 Pro
   - iPhone SE
   - Pixel 5
   - iPad

### 真机调试

**iOS (Safari):**
1. iPhone设置 → Safari → 高级 → 网页检查器（开启）
2. Mac Safari → 开发 → [你的iPhone] → 选择页面

**Android (Chrome):**
1. 手机开启开发者选项和USB调试
2. Chrome访问 `chrome://inspect`
3. 选择你的设备
