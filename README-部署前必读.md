## 🚀 快速解决方案（3选1）

### ⭐ 方案1：在线工具生成（最快）

1. 访问：https://www.pwabuilder.com/imageGenerator
2. 上传一个512x512的图片
3. 下载生成的图标包
4. 解压到 `assets/icons/` 文件夹

### 方案2：Emoji图标生成

1. 访问：https://favicon.io/emoji-favicons/tomato/
2. 选择🍅emoji
3. 下载并重命名文件

### 方案3：临时方案测试（无图标）

```bash
# 将临时配置复制为正式配置
copy manifest-minimal.json manifest.json
```

⚠️ 方案3只能用于测试，正式使用必须有图标！
