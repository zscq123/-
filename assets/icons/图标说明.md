# 应用图标说明

## 需要准备的图标尺寸

请在此文件夹中放置以下尺寸的PNG图标文件：

- `icon-72.png` - 72x72px
- `icon-96.png` - 96x96px
- `icon-128.png` - 128x128px
- `icon-144.png` - 144x144px
- `icon-152.png` - 152x152px
- `icon-192.png` - 192x192px ⭐ 重要
- `icon-384.png` - 384x384px
- `icon-512.png` - 512x512px ⭐ 重要

## 图标设计建议

### 风格
- 简洁现代
- 圆角矩形背景（iOS风格）
- 使用主色调 #007AFF
- 可包含番茄🍅或任务清单📝元素

### 颜色方案
```
主色：#007AFF (iOS Blue)
背景：白色或渐变
图标：简洁的番茄或勾选标志
```

### 示例设计

```
┌─────────────┐
│             │
│    🍅       │  背景：白色或蓝色渐变
│   📝✓       │  图标：番茄+勾选
│             │
└─────────────┘
```

## 临时解决方案

如果暂时没有设计好的图标，可以：

1. **使用在线图标生成器**
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. **使用Emoji作为临时图标**
   - 访问 https://favicon.io/emoji-favicons/
   - 选择🍅或📝表情
   - 下载并重命名为对应尺寸

3. **使用简单的纯色图标**
   - 创建纯蓝色背景
   - 中心放置白色番茄或勾选符号

## Apple Touch Icon

另外还需要一个 `apple-touch-icon.png` (180x180px) 用于iOS设备。

## 快速生成命令（需要ImageMagick）

如果你有一个512x512的源图标 `source.png`：

```bash
# 生成所有尺寸
convert source.png -resize 72x72 icon-72.png
convert source.png -resize 96x96 icon-96.png
convert source.png -resize 128x128 icon-128.png
convert source.png -resize 144x144 icon-144.png
convert source.png -resize 152x152 icon-152.png
convert source.png -resize 192x192 icon-192.png
convert source.png -resize 384x384 icon-384.png
convert source.png -resize 512x512 icon-512.png
convert source.png -resize 180x180 apple-touch-icon.png
```

---

**注意：** 在正式部署前，请确保所有图标文件都已生成并放置在此文件夹中。
