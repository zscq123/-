# ⚠️ 临时图标生成说明

## 当前状态
**图标文件缺失 - PWA无法正常安装**

## 快速解决方案（3选1）

### 方案1：使用在线工具生成（推荐）⭐

访问：https://www.pwabuilder.com/imageGenerator

1. 上传一个512x512的图片（可以是emoji截图）
2. 自动生成所有尺寸
3. 下载zip，解压到此文件夹

### 方案2：使用Emoji生成器

访问：https://favicon.io/emoji-favicons/tomato/

1. 选择🍅番茄emoji
2. 下载生成的图标包
3. 重命名并复制到此文件夹

### 方案3：临时禁用图标（测试用）

**修改 manifest.json，删除 icons 和 shortcuts 配置：**

```json
{
  "name": "勿蹉跎 - 专注Todo",
  "short_name": "勿蹉跎",
  "description": "轻量级移动端任务管理与番茄钟应用",
  "start_url": "./index.html",
  "display": "standalone",
  "background_color": "#F2F2F7",
  "theme_color": "#007AFF",
  "orientation": "portrait",
  "scope": "./"
}
```

⚠️ **注意：** 方案3只能用于测试，正式使用必须有图标！

## 需要的图标文件清单

- ✅ icon-72.png (72x72)
- ✅ icon-96.png (96x96)
- ✅ icon-128.png (128x128)
- ✅ icon-144.png (144x144)
- ✅ icon-152.png (152x152)
- ✅ **icon-192.png (192x192)** - 最重要
- ✅ icon-384.png (384x384)
- ✅ **icon-512.png (512x512)** - 最重要

**最低要求：** 必须有 192x192 和 512x512 两个尺寸

---

## 简易生成脚本（需要ImageMagick）

如果你有一个 source.png (512x512)：

```bash
# Windows (PowerShell)
magick source.png -resize 72x72 icon-72.png
magick source.png -resize 96x96 icon-96.png
magick source.png -resize 128x128 icon-128.png
magick source.png -resize 144x144 icon-144.png
magick source.png -resize 152x152 icon-152.png
magick source.png -resize 192x192 icon-192.png
magick source.png -resize 384x384 icon-384.png
magick source.png -resize 512x512 icon-512.png
```

## ⚡ 快速修复（启动前）

**在启动服务器测试前，必须先解决图标问题！**

否则会出现：
- ❌ PWA安装提示不出现
- ❌ 安装后图标显示错误
- ❌ 浏览器控制台报错

**推荐：使用方案1或方案2，5分钟搞定！**
