# 勿蹉跎（手机版）

> 轻量级移动端任务管理与番茄钟PWA应用

## 📱 项目简介

基于iOS设计规范的移动端任务管理应用，采用PWA技术实现，可安装到手机桌面，支持离线使用。

### 核心功能

✅ 任务管理（增删改查）  
✅ 任务分类（工作/生活/学习/健康）  
✅ 分类筛选  
✅ 任务搜索  
✅ 番茄钟计时器  
✅ 今日统计  
✅ 连续打卡  
✅ 本地存储  
✅ 暗色模式  
✅ PWA离线支持

## 🎨 设计特色

- 完全遵循**Apple Human Interface Guidelines**
- **毛玻璃效果**（Frosted Glass）
- **暗色模式**自动适配
- **Safe Area**适配（刘海屏/Home Indicator）
- 流畅的**iOS风格动画**
- 所有触摸区域≥44px

## 🚀 快速开始

### 本地运行

1. 克隆项目
```bash
git clone <项目地址>
cd 勿蹉跎（手机版）
```

2. 启动本地服务器
```bash
# 使用Python
python -m http.server 8080

# 或使用Node.js
npx serve

# 或使用Live Server (VS Code插件)
```

3. 在浏览器打开
```
http://localhost:8080
```

### 移动端测试

1. 确保手机和电脑在同一局域网
2. 查看电脑IP地址
3. 手机浏览器访问 `http://[电脑IP]:8080`

### 安装到手机

#### iOS (Safari)
1. 打开应用
2. 点击分享按钮
3. 选择"添加到主屏幕"
4. 点击"添加"

#### Android (Chrome)
1. 打开应用
2. 点击菜单（三个点）
3. 选择"安装应用"或"添加到主屏幕"
4. 点击"安装"

## 🎯 使用说明

### 添加任务
1. 点击右下角➕按钮
2. 输入任务内容
3. 选择分类
4. 点击"添加"

### 完成任务
- 点击任务卡片右下角的圆圈✓

### 删除任务
- 长按任务卡片800ms

### 启动番茄钟
- 点击任务卡片右上角的🍅按钮

### 分类筛选
- 点击顶部的分类按钮（🔵🟢🟡🔴）

### 搜索任务
- 在搜索框输入关键词

## 📊 数据存储

所有数据存储在浏览器的LocalStorage中，包括：
- 任务列表
- 番茄钟历史
- 成就数据
- 应用设置

**注意：** 清除浏览器数据会导致任务丢失！
